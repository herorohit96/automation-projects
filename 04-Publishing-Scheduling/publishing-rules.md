PUBLISHING RULES SYSTEM

1. Never publish raw content.
   → Content must pass formatting + caption check.

2. One platform = one custom version.
   → No direct copy-paste across platforms.

3. Priority order:
   a) Value
   b) Clarity
   c) Call-to-action

4. Posting frequency:
   - Reels / Shorts: Daily or alternate day
   - Carousel / Posts: 3–4 per week

5. Time rule:
   - Test 2 fixed time slots only.
   - Do not change timing daily.

6. CTA rule:
   Every post must contain ONE clear action:
   - Follow
   - Save
   - Comment
   - Link click
