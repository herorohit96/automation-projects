# 04 – Publishing & Scheduling System

This module defines WHERE, WHEN, and HOW content is published.

---

## PLATFORM PRIORITY (BEGINNER MODE)

1. Instagram (Primary)
2. YouTube Shorts (Secondary)
3. Telegram (Storage + Community)

Focus order must be followed strictly.

---

## DAILY CONTENT FLOW

1 Idea → 1 Reel → 1 Short → Telegram Post

Never create separate content for each platform.
Repurpose only.

---

## INSTAGRAM REELS – RULES

Posting Frequency:
- 1 Reel per day (minimum)
- Best: 5–6 days/week

Best Time (India):
- Morning: 9:00 AM – 11:00 AM
- Night: 8:00 PM – 10:00 PM

Caption Structure:
1. Hook line
2. Value summary (2–3 lines)
3. CTA (Save / Follow / DM / Telegram)

Hashtags:
- 3 niche hashtags
- 3 broad reach hashtags
- 2 location/India-based

---

## YOUTUBE SHORTS – RULES

Posting Frequency:
- Same reel reposted as Short
- 1 per day or alternate days

Best Time:
- Afternoon: 1:00 PM – 3:00 PM
- Night: 7:00 PM – 10:00 PM

Description:
- 1 value line
- 1 CTA
- Optional hashtag

---

## TELEGRAM – STORAGE SYSTEM

Purpose:
- Store content
- Build trust
- Distribute PDFs & links

Posting:
- Same reel caption (text only)
- Carousel images
- Free resources

Telegram Rule:
Telegram is NOT for virality.
Telegram is for retention + monetization.

---

## CTA ROTATION SYSTEM

Do NOT use same CTA daily.

Rotate between:
- Save this post
- Follow for daily AI tips
- Comment "PDF"
- Join Telegram
- DM for guide

---

## WEEKLY SCHEDULE (BEGINNER SAFE)

Monday – Reel (Education)
Tuesday – Reel (Tip)
Wednesday – Reel (Problem + Solution)
Thursday – Reel (Tool based)
Friday – Reel (Motivation / Proof)
Saturday – Optional
Sunday – Review / Learn

---

## GOLDEN RULES

- Consistency > Perfection
- Same content everywhere
- Never skip CTA
- Track what works weekly

This module works only if Content Production module is followed.
