# n8n Automation System

This system manages n8n workflows using Docker only.

## Structure

- 01_SETUP: Installation and setup documentation
- 02_WORKFLOWS: Workflow logs and exported JSON files
- 03_CHANGES_LOG: Daily log of all actions
- 04_BACKUP: Backup instructions
- 05_TROUBLESHOOTING: Common errors and fixes

All actions are logged in the changes log.
