# What is n8n?

## n8n in Simple Words

n8n is an automation tool that helps you connect apps, tools, and systems without coding.

Simple language me:
n8n = digital worker jo tumhare rules follow karta hai.

---

## Why n8n is Powerful

Most tools sirf ek kaam karte hain.
n8n poora system banata hai.

You can:
- Connect multiple apps
- Create logic-based workflows
- Automate boring & repetitive tasks
- Run everything in background

---

## How n8n Actually Works

Every n8n workflow follows this logic:

Trigger → Condition → Action → Output

Example:
Form submit → Check data → Send WhatsApp → Save in Google Sheet
