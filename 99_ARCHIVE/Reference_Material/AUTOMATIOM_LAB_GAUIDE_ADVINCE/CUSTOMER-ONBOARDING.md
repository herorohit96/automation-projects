## COMMON ISSUES & SOLUTIONS

### Issue 1: Workflow Not Running
**Solution:**
1. Check n8n is running
2. Check all nodes have green checkmarks
3. Verify credentials are correct
4. Try running test execution

### Issue 2: No Transactions
**Solution:**
1. Check you have traffic
2. Check targeting is correct
3. Wait 1-2 weeks (sometimes takes time)
4. Review FAQ section on optimization

### Issue 3: Google Sheets Error
**Solution:**
1. Re-authenticate Google account
2. Share spreadsheet with service account email
3. Check column names match workflow

### Issue 4: Telegram Not Posting
**Solution:**
1. Check bot token is correct
2. Check channel is public
3. Check channel ID is correct
4. Verify bot has permission to post
