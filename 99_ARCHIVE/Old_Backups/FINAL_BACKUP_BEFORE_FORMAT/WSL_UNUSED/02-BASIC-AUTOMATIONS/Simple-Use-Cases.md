# Simple Use Cases

Yahan 6 simple automations hain jo aap bana sakte hain. Har ek ko step-by-step implement karo.

## 1. Email Auto-Reply
Jab koi email aaye, automatic reply bhejo.
- Trigger: New email in Gmail
- Action: Send predefined reply
- Use case: Business hours ke baad queries handle karna

## 2. Lead Capture from Form
Google Form se data ko Sheet mein save karo.
- Trigger: Form submission
- Action: Add to Google Sheet, send thank you email
- Use case: Website se leads collect karna

## 3. Daily Backup Reminder
Har din reminder bhejo data backup ke liye.
- Trigger: Daily at 5 PM
- Action: Send Telegram message
- Use case: Personal ya business data safety

## 4. Social Media Post Scheduler
Posts ko schedule karo automatic post karne ke liye.
- Trigger: Time-based
- Action: Post to platform
- Use case: Content marketing automation

## 5. Invoice Reminder
Pending invoices ke liye reminder send karo.
- Trigger: Weekly on Monday
- Action: Check database, send emails
- Use case: Payment collection improve karna

## 6. Weather Alert
Bad weather mein alerts bhejo.
- Trigger: Weather API data
- Action: Send Telegram notification
- Use case: Event planning ya safety alerts

Har automation ko banao, test karo, aur optimize karo. Yeh aapke portfolio ke liye good examples hain.
