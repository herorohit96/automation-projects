# 02 – Content Production System

This module is responsible for generating DAILY content ideas
for a faceless digital brand.

## RULES
- Target audience: beginners, middle-class, zero-knowledge users
- Language: Simple Hinglish
- Platform-first thinking (Instagram, YouTube Shorts)
- No face, no personal branding
- Focus on value + motivation + clarity

---

## PROMPT 1: DAILY CONTENT IDEAS

Use this prompt to generate content ideas.

Prompt:
"You are a content strategist for a faceless digital brand.
Generate 10 content ideas for beginners related to:
[TOPIC]

Each idea must include:
- Content angle
- Platform (Reel / Short / Post)
- One-line hook"

---

## PROMPT 2: REEL / SHORT SCRIPT

Prompt:
"Write a 30–45 second faceless reel script for beginners on:
[TOPIC]

Structure:
- Hook (first 3 seconds)
- Problem
- Simple explanation
- Clear takeaway
- Soft CTA

Language: Hinglish
Tone: Friendly mentor"

---

## PROMPT 3: POST CAPTION

Prompt:
"Write an Instagram caption for a faceless post on:
[TOPIC]

Include:
- 2-line hook
- Simple explanation
- Bullet points
- CTA to save/share"

---

## PROMPT 4: HOOK GENERATOR

Prompt:
"Generate 10 viral hooks for beginner-focused content on:
[TOPIC]

Hooks should:
- Create curiosity
- Be simple
- Avoid technical jargon"

---

## USAGE INSTRUCTION

1. Pick ONE topic
2. Run Prompt 1
3. Select one idea
4. Use Prompt 2 or 3
5. Publish manually or via automation later

This module must always follow the Master System Prompt.
