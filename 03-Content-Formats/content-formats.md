# 03 – Content Formats System

This module defines HOW content is structured and presented.

---

## FORMAT 1: INSTAGRAM REEL (30–45 sec)

Structure:
1. Hook (0–3 sec)
2. Problem (3–7 sec)
3. Explanation (7–25 sec)
4. Key takeaway (25–35 sec)
5. CTA (last 5 sec)

Rules:
- No face
- Text + voice or text-only
- One idea per reel

---

## FORMAT 2: YOUTUBE SHORT

Structure:
1. Strong curiosity hook
2. Fast explanation
3. One clear learning
4. CTA (subscribe/save)

Rules:
- Faster pacing than reels
- No fillers
- Beginner friendly

---

## FORMAT 3: CAROUSEL POST

Slide Structure:
- Slide 1: Big hook
- Slide 2–4: Problem + explanation
- Slide 5–6: Steps / tips
- Last slide: CTA

Rules:
- Simple language
- Big readable text
- One topic only

---

## FORMAT 4: STATIC POST

Structure:
- Headline
- 3–5 bullet points
- CTA

Best for:
- Motivation
- Simple tips
- Awareness content

---

## USAGE RULE

1. Take idea from Content Production module
2. Select ONE format
3. Apply structure
4. Publish

This module must always follow the Master System Prompt.
